# ============================================================================
# DEPENDENCY NOTES
# ============================================================================

# Version Philosophy:
#   - Minimum versions specified (>=) allow flexibility
#   - Versions tested: Python 3.8, 3.9, 3.10, 3.11, 3.12
#   - Core dependencies kept minimal for broad compatibility

# Why minimal new dependencies in v2.2.x:
#   - Modules 7, 8, 9 designed to use existing scipy, numpy, pandas
#   - Module 6b (Acquisition) adds requests + pyarrow as core,
#     with GEOparse/biopython/mygene as optional per-connector deps
#   - Ensemble methods use standard statistical functions
#   - Parameter optimization uses scipy.optimize

# External Tool Requirements (NOT Python packages):
#   - R 4.0+ with packages: DESeq2, edgeR, limma (for Module 6)
#   - STAR, HISAT2, Kallisto, Salmon (for pipelines)
#   - Samtools, featureCounts, RSEM (for pipelines)
#
#   Install R packages:
#     Rscript raptor/external_modules/module6_de_analysis/r_scripts/install_packages.R

# Platform-specific notes:
#   - All core dependencies are cross-platform (Windows, Linux, macOS)
#   - Dashboard requires modern browser for Streamlit
#   - PDF generation (weasyprint) may require system libraries


# ============================================================================
# TROUBLESHOOTING
# ============================================================================

# If installation fails:
#   1. Update pip: python -m pip install --upgrade pip
#   2. Update setuptools: pip install --upgrade setuptools wheel
#   3. Try installing individual packages to identify the issue
#   4. Check Python version: python --version (need 3.8+)

# If specific packages fail:
#   - statsmodels: May require compiler on some systems
#   - matplotlib: May require system libraries for rendering
#   - scipy: May require BLAS/LAPACK libraries

# Virtual environment (strongly recommended):
#   python -m venv raptor_env
#   source raptor_env/bin/activate  # Linux/Mac
#   raptor_env\Scripts\activate     # Windows
#   pip install -r requirements.txt
